Scenario samples for Upshot
Scenario Description
Simple get Demonstrates how to use upshot.RemoteDataSource to retrieve list data asynchronously via HTTP GET and visualize the result in an HTML table.
Page, sort and filter Building upon the previous sample, this sample demonstrates how to add server-side paging, sorting and filtering to a displayed list of data.
In-place property editing This sample demonstrates simple, in-place editing of data entities loaded from a service. Adjust the rating of products as they’re displayed in a table.
Batched editing This sample demonstrates editing over multiple data entities, where edits are submitted to a service in a batch.
List add/remove This sample demonstrates adding and removing data entities from a list.
